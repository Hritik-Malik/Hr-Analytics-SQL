# Project Summary

This project demonstrates a simulated HR analytics system built using a relational SQL database. It focuses on generating key HR metrics and insights to support workforce and payroll analysis.

# Database Structure

The project consists of the following tables:

Departments

Employees

Attendance

Payroll

# Key Insights Generated

Complete employee overview report

Department-level employee count

Average compensation by department

Individual attendance percentage

Payroll expenses and net salary analysis

Salary ranking across employees

HR KPI summary

# Technologies Used

SQL Server Management Studio (SSMS)

GitHub

# Execution Steps

Run 01_data.sql to create the database and populate tables.

Run 02_showdata.sql to validate the inserted records.

Run 03_queries.sql to perform analytical queries and generate insights.
